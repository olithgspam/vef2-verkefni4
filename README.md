# Vefforritun 2 - Verkefni 4: Fréttavefurinn

Nútímalegur fréttavefur byggður á Next.js 15, hannaður með áherslu á hraða, öryggi og notendaupplifun.

Hægt er að skoða verkefnið í vinnslu á: [oli.gs](https://oli.gs)

## Helstu eiginleikar
- **Dynamic Content:** Fréttir og höfundar sóttir úr ytra API (verkefni.gold).
- **Admin Shield:** Lykilorðavörn (132132) á öllum eyðingaraðgerðum.
- **Server Actions:** Örugg samskipti við bakenda (POST/DELETE) sem komast framhjá CORS-takmörkunum.
- **Next.js 15 Routing:** Notkun á async params í dýnamískum leiðum (/[slug]).
- **Umsjónarkerfi höfunda:** Sérsíða (/hofundar) til að halda utan um fréttaritara.
- **Modern UI:** Hannað með Tailwind CSS, gler-áhrifum (backdrop-blur) og lifandi bakgrunni.

## Tæknistafur (Tech Stack)
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **State Management:** React Hooks (useState, useEffect)
- **Backend:** Ytra REST API

## Admin upplýsingar
Til að eyða fréttum eða höfundum þarf að nota eftirfarandi lykilorð:
- **Lykilorð:** 132132

## Keyrsla í þróunarumhverfi
1. Keyra `npm install` til að sækja pakka.
2. Keyra `npm run dev` til að ræsa serverinn á `localhost:3000`.
